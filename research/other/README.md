# Other Materials

This folder contains supplementary materials that support the research but don't fit into `linkedin-posts/` or `youtube-transcripts/` — for example:

- Newsletter issues (e.g. Kevin Indig's Growth Memo, Aleyda Solis's #SEOFOMO)
- Podcast episode notes
- Articles published directly on company blogs (e.g. Ahrefs blog posts by Ryan Law)
- Reddit threads (e.g. from r/AISEOforBeginners, moderated by Oleg/dashosh)

Add files here as `[author-name]-[short-title].md`, and note the source URL at the top of each file.
